## Prova da Materia de Programação de Dispositivos Mobile

- Prova consiste em fazer uma interface grafica que irá adicionar um aluno e suas informações, logo após mostrar uma lista com todos os alinos, se caso a nota for abaixo de 6 irá mostrar em vermelho o aluno
- Se caso clicar no aluno irá mostrar as informações do aluno na tela
